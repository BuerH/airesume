# Security

`airesume` reads local AI coding assistant session files and prints commands that can resume those sessions.

## Data Handling

- Session files are read-only.
- `airesume` does not upload session data.
- AI summaries are not implemented yet. When summary providers are added, they must be opt-in and documented clearly.

## Reporting Issues

Please open a private security advisory on GitHub if the repository has advisories enabled. Otherwise, open an issue with minimal reproduction details and avoid pasting private session contents.
